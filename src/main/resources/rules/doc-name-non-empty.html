Component doc:name must not be empty

All components with a doc:name attribute must provide a meaningful non-empty value. Empty names make flow diagrams unreadable.

Noncompliant Code

<set-variable variableName="customerId" value="#[payload.id]" doc:name=""/>

Compliant Code

<set-variable variableName="customerId" value="#[payload.id]" doc:name="Extract Customer ID"/>
