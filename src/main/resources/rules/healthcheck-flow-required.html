Application must include a health check flow

Application must contain at least one flow with 'healthcheck' in its name. Health check endpoints are essential for monitoring, load balancers, and orchestration platforms to verify application availability and readiness.

Noncompliant Code

<mule xmlns="http://www.mulesoft.org/schema/mule/core">
  <flow name="process-data"><logger/></flow>
</mule>

Compliant Code

<mule xmlns="http://www.mulesoft.org/schema/mule/core">
  <flow name="healthcheck-api"><logger/></flow>
  <flow name="process-data"><logger/></flow>
</mule>
