Configuration file exceeds maximum flow count limit

Configuration files should contain fewer than 10 flows to maintain readability and modularity. Consider splitting large configuration files into multiple smaller files organized by domain or functionality.

Noncompliant Code

<mule xmlns="http://www.mulesoft.org/schema/mule/core">
  <flow name="flow-1"><logger/></flow>
  <flow name="flow-2"><logger/></flow>
  <flow name="flow-3"><logger/></flow>
  <flow name="flow-4"><logger/></flow>
  <flow name="flow-5"><logger/></flow>
  <flow name="flow-6"><logger/></flow>
  <flow name="flow-7"><logger/></flow>
  <flow name="flow-8"><logger/></flow>
  <flow name="flow-9"><logger/></flow>
  <flow name="flow-10"><logger/></flow>
</mule>

Compliant Code

<mule xmlns="http://www.mulesoft.org/schema/mule/core">
  <flow name="flow-1"><logger/></flow>
  <flow name="flow-2"><logger/></flow>
  <flow name="flow-3"><logger/></flow>
</mule>
