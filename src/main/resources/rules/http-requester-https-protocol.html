HTTP Request must use HTTPS protocol for secure communication

HTTP Request configuration must specify protocol='HTTPS' in . Using HTTP for outbound requests exposes data in transit to interception. HTTPS encrypts all communication with external services.

Noncompliant Code

<mule xmlns="http://www.mulesoft.org/schema/mule/core"
      xmlns:http="http://www.mulesoft.org/schema/mule/http">
  <http:request-config name="HTTP_Request_config">
    <http:request-connection host="api.example.com" port="80" protocol="HTTP"/>
  </http:request-config>
</mule>

Compliant Code

<mule xmlns="http://www.mulesoft.org/schema/mule/core"
      xmlns:http="http://www.mulesoft.org/schema/mule/http">
  <http:request-config name="HTTP_Request_config">
    <http:request-connection host="api.example.com" port="443" protocol="HTTPS"/>
  </http:request-config>
</mule>
